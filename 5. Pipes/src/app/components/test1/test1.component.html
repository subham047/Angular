Pipes Concepts

**Pipe Operator transfers the data only for the view. It does not change the value of the property in the class**

Pipes Conecpt with properties

{{name}}

{{name | uppercase}}

{{name | lowercase}}

{{name | titlecase}}

{{name | slice : 3}}

{{name | slice : 3:12}}

Pipes Conecpt with Json

{{person | json}}

Pipes Conecpt with numbers. Use of number pipe

{{5.678 | number : '1.2-3'}}

{{5.678 | number : '3.4-5'}}

{{5.678 | number : '3.1-2'}}

Percent pipe

{{0.25 | percent}}

Currency pipe

{{0.25 | currency}}

Date pipe

{{date}}

{{date | date:'short'}}

{{date | date:'shortDate'}}

{{date | date:'shortTime'}}
